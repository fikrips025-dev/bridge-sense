Metodologi & Parameter
📐 SCS → ECS Projection

Akselerometer smartphone merekam getaran dalam Smartphone Coordinate System (SCS) yang bergantung orientasi fisik perangkat. Bridge-Sense memproyeksikannya ke Earth Coordinate System (ECS) via matriks rotasi. Hanya komponen vertikal Z yang diteruskan ke buffer.

a_ECS = R(α,β,γ) · a_SCS
a_linear = a_total − g_vector
g_vector ≈ [0, 0, 9.80665] m/s²

Izin Sensor Gerak

Bridge-Sense memerlukan akses ke akselerometer dan giroskop perangkat untuk mengukur getaran jembatan secara real-time.

⚠️ Pada iOS 13+, izin sensor gerak harus diminta secara eksplisit. Data sensor tidak pernah dikirim tanpa konfirmasi.
